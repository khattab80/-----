# 🏛️ بوت أثر الأنثروبولوجي (Athar Anthropology Bot)

بوت تليجرام ذكي ومستقل مخصص لقناة **[أثر | علم الإنسان](https://t.me)**. يعتمد البوت على تقنيات الذكاء الاصطناعي عبر **Groq API** لتقديم محتوى متجدد وموثوق حول علم الإنسان (الأنثروبولوجيا)، بالإضافة إلى التفاعل التلقائي في مجموعة التعليقات لإثراء النقاش العلمي.

---

## ✨ المميزات الرئيسية

* **نشر تلقائي ودوري:** يقوم بنشر منشور علمي جديد كل 30 دقيقة بشكل متجدد وغير متكرر.
* **مصادر بيانات موثوقة:** يجلب معلوماته الخام من موسوعات ومصادر علمية موثوقة (Wikipedia API).
* **إعادة صياغة ذكية:** يعيد صياغة المعلومات بأسلوب "عالم إنثروبولوجيا خبير ومتحدث لبق بالفصحى" عبر نموذج Llama 3 المدعوم من Groq.
* **منشورات تفاعلية ومسجوعة:** يقسم المنشور إلى نقاط مشوقة مدعومة بالرموز التعبيرية ويختمها بسؤال تفاعلي.
* **صور متجددة عالية الجودة:** يرفق مع كل منشور صورة متغيرة وعالية الدقة تعبر عن الكلمة المفتاحية للموضوع عبر **Unsplash API**.
* **الرد الذكي في التعليقات:** يراقب مجموعة التعليقات المرتبطة بالقناة، وعند تحويل أي منشور إليها، يدخل فوراً ليضيف معلومة تاريخية أو ثقافية حصرية وثريّة دون تكرار محتوى المنشور.

---

## 📁 هيكلة ملفات المشروع

يتكون المشروع من 3 ملفات رئيسية فقط جاهزة للرفع مباشرة على سيرفر التشغيل:
1. `bot.py`: الكود البرمجي الأساسي للبوت بلغة بايثون.
2. `requirements.txt`: ملف المكتبات البرمجية المطلوبة للتشغيل.
3. `Procfile`: ملف تعريف بيئة التشغيل المستمر على منصات السحاب مثل Railway.

---

## 🛠️ متطلبات الإعداد قبل التشغيل

لضمان عمل البوت بكفاءة، يجب توفير المتغيرات والمفاتيح التالية:
1. **توكن البوت (Telegram Bot Token):** يتم استخراجه من الـ BotFather.
2. **مفتاح جروق (Groq API Key):** يتم استخراجه مجاناً من منصة Groq Console.
3. **مفتاح أنسبلاش (Unsplash Access Key):** يتم استخراجه مجاناً للمطورين لجلب الصور تلقائياً.

### ⚙️ إعدادات التليجرام الهامة:
* قم برفع البوت كمشرف (**Admin**) في قناتك `@Athar_Anthro` مع تفعيل صلاحية نشر المنشورات.
* قم برفع البوت كمشرف (**Admin**) في مجموعة النقاش/التعليقات المرتبطة بالقناة مع تفعيل صلاحية قراءة الرسائل والرد عليها.

---

## 🚀 طريقة التشغيل على سيرفر Railway

البوت مهيأ بالكامل للتعرف على متغيرات البيئة تلقائياً عند رفعه على **Railway**:

1. قم برفع المجلد الذي يحتوي على الملفات الثلاثة (`bot.py`, `requirements.txt`, `Procfile`) إلى مستودع خاص بك على **GitHub**.
2. اربط مستودع GitHub بمنصة **Railway** عبر إنشاء مشروع جديد (New Project -> Deploy from GitHub).
3. توجه إلى قسم المتغيرات (**Variables**) في مشروعك على Railway وأضف المفاتيح التالية بالتسميات التالية تماماً:

| اسم المتغير (Variable Name) | القيمة (Value) |
| :--- | :--- |
| `TELEGRAM_BOT_TOKEN` | توكن البوت الخاص بك من BotFather |
| `GROQ_API_KEY` | مفتاح الـ API الخاص بك من Groq |
| `CHANNEL_ID` | معرف قناتك (مثال: `@Athar_Anthro`) |

4. بمجرد إضافة المتغيرات، سيبدأ السيرفر بالعمل تلقائياً وبشكل مستمر (24/7) دون أي تدخل منك.
